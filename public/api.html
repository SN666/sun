红色为必填参数
1.会员注册【post】
参数
account(账号)
mobile(手机号)
password(密码明文)
realname(真实姓名)
idcard(身份证号)
country(国家)
prov(省)（传汉字）
city(市)（传汉字）
area(区)（传汉字）
village(村)（传汉字）
address(详细地址)（传汉字）
sd(sd号)
地址：/index.php/api/user/reg
返回：
{
  "code": "200",
  "msg": "成功"
}
